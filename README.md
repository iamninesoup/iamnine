

## 설명
- 이 프로젝트는 Arduino Pro Micro를 이용한 미니 MIDI 컨트롤러 코드입니다.
- 버튼/조이스틱/슬라이더를 활용하여 DAW와 연동 가능

## 기능
- 4~6 버튼 입력
- MIDI 신호 전송
- (추가: 조이스틱, 노브 등 지원)

## 설치 및 사용 방법
1. Arduino IDE 설치
2. 코드 다운로드 및 열기
3. 필요한 라이브러리 설치 (예: MIDI.h)
4. Arduino Pro Micro에 업로드
